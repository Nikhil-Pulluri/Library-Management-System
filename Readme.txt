Library Management System

This is a simple command-line based Library Management System implemented in C. The system allows users to add books, add students, issue books to students, return books, and check the status of a book.

Features

1. Add Book:

Adds a new book to the library with a unique book ID, name, and author.

2. Add Student:

Adds a new student to the system with a unique student ID and name.

3. Issue Book:

Allows a student to borrow a book. Students are limited to borrowing up to 3 books.

4. Return Book:

Allows a student to return a previously issued book.

5. Show Student Books:

Displays a list of books issued to a specific student.

6. Check Book Status:

Checks whether a particular book is currently issued or available.

7. Exit:

Exits the program.
